## Installation (Swift Package Manager)
Add the following entry to your package dependencies...
```swift
.package(url: "https://github.com/ryanslikesocool/SwiftyAccessibility.git", from: "0.0.1"),
```
...and your target dependencies.
```swift
.target(
	name: "MyTarget",
	dependencies: [
		"SwiftyAccessibility",
	]
),
```
